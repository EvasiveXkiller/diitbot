A very warm welcome to users of International of DIIT Congress Discord bot. The DIIT bot is a personal project led by EvasiveXkiller with his teammates Araric, ikennot and XianXian. The creation of DIIT bot started during late 2020 and this bot is intended to be a bot that entertains and keeps memories of every single member of the private server.

Functionalities and Types of Commands

    • Avatar

    • $avatar is a command which allows user to view his/her profile picture. When $avatar used, the sender's profile picture will be displayed. When the sender send $avatar @userExample, @userExample's profile picture will be displayed.

    • Profile

    • $profile is a command which allows user to view his/her profile information which consist Facebook, Instagram and etc. $profile has to have a name attached with command, for example $profile userName, then all information of userName's social will be displayed. Otherwise, nothing will be displayed if $profile only is sent

    • Iconic

    • $iconic is a command which allows user to view iconic quotes given by different members of the server. $iconic can have or not have a name attached with the command, for example, if sender send $iconic, the bot will display quotes given by any members of the server randomly. If $iconic is attached with a name, for example $iconic James, then the bot will only display quotes given by James only.

    • Add a New Quote

    • $dbinsert is a command which allows user to add a new quote. When $dbinsert command is prompted, instruction will be given to the user in order to add new quotes.User will have to insert name first then the quote to add. Note that there's time limit of 15 seconds before the required informations are gathered. After 15 seconds, information will be gathered whether you have inputted or not and confirmation message will be prompted. If you want to add new quote, send "accept", otherwise send "cancel".

    • Deleting an Existing Quote

    • $dbdelete is a command which allows user to remove an existing quote. In order to delete a quote from a specific member, 2 important message must be provided to satisfy the parameters. In short, if you wanted to remove a quote such as "Hey" from James, type $dbdelete James,Hey and the quote will be permanently removed from the database.

    • Ping

    • $ping is a command which allows user to check the bot's latency

    • Help

    • $help is a command which could help new users learn the basic commands and function of this amazing bot.

    • Description

    • $description is a command which displays a brief introduction of this amazing bot.

    • Reset

    • $reset %reset% is a command which is used to reset the bot whenever bot no longer responds to commands.

    • Gif

    • $gif is a command which displays Gifs of a specific individual. In order for the bot to display Gif, a keyword is needed for the bot to search relevant gif. If $gif Jack is sent, gifs related to Jack will be shown. If only $gif is sent without keyword, error message will be prompted

    • Check Timetable

    • $timetable is a command which displays timetable of the week. $timetable command must have a weekday attached to it, for example $timetable Monday and all events of Monday will be shown as well as the time and duration of all events. "Today", "Tomorrow" and "Yesterday" can also be accepted and precisely show the accurate information of the event name and timing. In addition, if user entered $timetable all, all events from Monday to Friday will be displayed.

    • Vote Instruction

    • $vote is a command which shows instructions on how to create a voting poll.

    • Create Voting Poll

    • $voteinnit is a command which creates a voting poll.

    • Check Events

    • $events is a command which allows user to check all events stored in the bot's database

    • Memes

    • $memes is a command which display random memes retrieved from the bot's database

Tic Tac Toe

    • Tic Tac Toe Instruction

    • $ttt is a command which shows instructions on how to create a tic tac toe game.

    • Tic Tac Toe

    • !ttt is a command which creates a tic tac toe game. !ttt has to have a tagged user in the discord server attached with the command, for example !ttt @Jack.

UNO

    • UNO Instruction

    • $uno is a command which shows the basic instructions and commands to play the game.

    • UNO Commands

    • $unocreate Create a new uno game

    • $unojoin Join the created uno game

    • $unostart Start the uno game

    • $unoplay Play your cards

    • $unodraw Draw a new card if unable to play

    • $uno! Protect yourself from UNO!

    • $unoviewself View your current cards

    • $unoviewtable View what is on the table

    • $unoend Ends the current game

    • $unosettings Configure game settings

    • $unoviewsetting View Game Settings

Music

    • Play

    • $play, $p are commands which allows user to play a song or queue a song. $play usually has to attach a Youtube link or a song name, for example $play Happy Birthday

    • Volume

    • $volume is a command which allows user to set and adjust volume of the sound output. $volume usually attach a number ranging from 0-200 for best experience, for example $volume 100

    • Stop

    • $stop is a command which allows user to stop the bot from playing music and all queued songs will be removed.

    • Skip

    • $skip is a command which allows user to skip current playing song.

    • Queue

    • $queue is a command which allows user to view the whole queue of songs.

    • Remove Queue

    • $remove is a command which allows user to remove specific song in queue. For example, if user wanted to remove 3rd song in queue, he/she shall type $remove 3

    • Repeat Song

    • $repeatsong is a command which allows user to loop a song.

    • Repeat Queue

    • $repeatqueue is a command which allows user to loop a queue.

    • Disconnect

    • $dis, $dc, $fuckoff, $leave are commands which allows user to disconnect the bot from the voice channel.
